# 人性化工具

一个Clawdbot技能，用于去除文本中AI生成的痕迹，使其听起来更自然、更像人类写作。

## 安装

通过ClawdHub安装：

```bash
clawdhub install humanizer
```

## 使用方法

让你的代理人性化文本：

```
请人性化这段文本：[你的文本]
```

或在编辑文档时直接调用。

## 概述

基于[维基百科的"AI写作迹象"](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing)指南，由WikiProject AI Cleanup维护。这份全面的指南来自对数千个AI生成文本实例的观察。

### 关键见解

> "LLM使用统计算法来猜测接下来应该是什么。结果倾向于适用于最广泛情况的统计上最可能的结果。"

## 检测到的24种模式

### 内容模式
1. **重要性夸大** - "标志着关键时刻..." → 具体事实
2. **知名度炫耀** - 在没有上下文的情况下列出来源
3. **肤浅的-ing分析** - "象征着...反映着..."
4. **促销性语言** - "坐落在令人惊叹的..."
5. **模糊的归因** - "专家认为..."
6. **公式化的挑战** - "尽管有挑战...继续蓬勃发展"

### 语言模式
7. **AI词汇** - "此外...证明...格局..."
8. **系动词回避** - "作为"而不是"是"
9. **否定并列** - "这不仅仅是X，而是Y"
10. **三法则** - 强制将想法分成三组
11. **同义词循环** - 过度的同义词替换
12. **虚假范围** - 在无意义的尺度上"从X到Y"

### 风格模式
13. **破折号过度使用**
14. **粗体过度使用**
15. **内联标题列表**
16. **标题大小写**
17. **表情符号装饰**
18. **弯引号**

### 沟通模式
19. **聊天机器人痕迹** - "希望这有帮助！"
20. **截止日期免责声明** - "虽然细节有限..."
21. **谄媚语气** - "好问题！"

### 填充和模糊表达
22. **填充短语** - "为了"、"由于...的事实"
23. **过度模糊表达** - "可能潜在地可能"
24. **通用结论** - "未来看起来光明"

## 完整示例

**之前（AI腔调）：**
> 新的软件更新作为公司致力于创新的证明。此外，它提供了无缝、直观和强大的用户体验——确保用户可以高效地完成他们的目标。

**之后（人性化）：**
> 软件更新添加了批处理、键盘快捷键和离线模式。来自测试用户的早期反馈是积极的，大多数人报告任务完成速度更快。
